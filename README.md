# 🌧️ Rainfall Prediction using Machine Learning

This project focuses on predicting rainfall using historical weather data and machine learning models. It demonstrates the complete pipeline from data preprocessing to model evaluation.

# 📌 Overview
The goal of this project is to build a predictive model that can determine whether it will rain based on weather conditions.

# ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- XGBoost  
- CatBoost  

# 📊 Project Workflow
1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Model Building  
6. Model Evaluation  

# 🤖 Models Used
- Support Vector Machine (SVM)  
- Random Forest  
- XGBoost  
- CatBoost  


# 📈 Results
- Compared multiple models using Accuracy and AUC score  
- Random Forest and CatBoost showed strong performance  
- Model performance varied slightly based on tuning and features
